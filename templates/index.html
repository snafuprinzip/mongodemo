MongoDB Demo for OpenShift

This is a small demo program to demonstrate the use of a MongoDB database within an OpenShift Cluster environment.
You can add song titles if you want and the items will be stored in another MongoDB Pod within the same namespace as the application.
Database Credentials will be transferred via environment variables.

Have fun!
